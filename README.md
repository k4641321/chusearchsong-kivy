# 此项目为[chusearchsong](https://github.com/k4641321/chusearchsong)kivy的重构版

# 中二查歌

一个使用kivy框架写的查歌软件，歌曲数据来源于落雪咖啡屋，目前为方便调试，歌曲数据是保存在本地，未来开发差不多时会换成联网获取

# 功能
- [x] 基本查歌
- [x] 分类查歌
- [x] 版本查歌
- [x] 定数筛选
- [ ] 歌曲详情
- [ ] 曲绘预览
- [ ] 收藏夹
- [ ] 歌曲试听
- [ ] 别名查歌
- [ ] 多个收藏夹
